# ArtNet — Expo / React Native project

This is an **Expo SDK 54 + React Native** project using Expo Router v3.
It is NOT a Next.js project.

- Do NOT suggest `"use client"` directives — they are a Next.js concept and do not apply here.
- Do NOT apply Next.js-specific validations or patterns.
- All `.tsx` files are React Native components, not Next.js pages or Server Components.
- The web build is a single-page app (SPA) exported via `expo export -p web` and deployed to Vercel as static files.
